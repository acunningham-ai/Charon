Install

Clone it, read it, run it, cross.

Charon runs on the Claude Code CLI. One command after cloning detects what you have, offers to install what you don't, then hands off to a ~20-minute first-run wizard that builds your foundations.

Claude Code CLI Python 3.10+ Obsidian optional

Four steps

The install, honestly.

No magic one-liner that pipes a script straight into your shell. Charon is a CISO tool — it asks you to read the installer before you run it. That step is the point.

1

Clone the repo

Anywhere — outside cloud-synced folders is faster. It's public and MIT.

2

Inspect the installer

Open install.ps1 / install.sh and skim it — it's short, and it tells you which package managers it'll invoke and where it'll write. Modelling the discipline the tool teaches starts here.

3

Run the bootstrap

It detects Python 3.10+ and Obsidian, offers (a)uto / (m)anual / (s)kip per missing prerequisite, installs Python dependencies, and creates ~/.secrets/ with restricted permissions.

4

Cross — the first-run wizard

The bootstrap hands off to a ~20-minute interactive setup that builds your identity, vault, voice profile, and framework. Then point Claude Code at the folder and start a session.

Copy-paste

The real commands.

macOS · Linux
# clone anywhere outside a cloud-synced folder
$ git clone https://github.com/acunningham-ai/Charon.git "$HOME/second-brain"
$ cd "$HOME/second-brain"

# read it first, then run the bootstrap
$ bash install.sh
Windows · PowerShell
# clone
$ git clone https://github.com/acunningham-ai/Charon.git "$env:USERPROFILE\second-brain"
$ cd "$env:USERPROFILE\second-brain"

# least-permissive policy that works for a locally-cloned script
$ powershell -ExecutionPolicy RemoteSigned -File install.ps1

Optional: every tagged release publishes a SHA-256 of the installer in its notes — Get-FileHash install.ps1 -Algorithm SHA256 to verify before running.

The first-run wizard

About 20 minutes, and you can stop any time.

The wizard walks five phases of questions — most always asked, the rest conditional on your answers. It seeds your research beats and forums so the standing seats produce value on day one, and scaffolds the full folder skeleton so every capability has a home.

State persists to ~/.charon-first-run-state.json — Ctrl+C any time and resume later. Re-run any phase on its own to refresh that section without redoing the rest.

02Voice profile

04Dashboard & tool exceptions
